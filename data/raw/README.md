# Dataset Documentation — Bengaluru House Price Data

Source:
Kaggle — Bengaluru House price data

File:
Bengaluru_House_Data.csv

License:
CC0 (Public Domain)

Target Variable:
price

Assumptions to Verify During EDA:

- Price unit ( lakhs INR)
- total_sqft contains mixed formats (e.g., ranges like "2100 - 2850")
- size column contains text values (e.g., "2 BHK", "4 Bedroom")
- location likely has high cardinality
- Missing values expected in society and possibly other columns

Notes:
Raw dataset is not modified. Cleaning will happen in data/processed later.
